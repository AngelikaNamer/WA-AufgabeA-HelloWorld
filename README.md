# Prerequisites
  - Windows Computer
  - latest JDK pack
  - IDE for Java development for example IntelliJ
  

# Installation
  - Open the Project in the IntelliJ IDE
  - Compile and transfer the program 
  
  ***or run Java code from a command line*** 
  
  ```
  java HelloWorld
  ```
